# AWS 인프라 구축
  클라우드 인프라의 경우 CSP에서 제공하는 웹UI나 CLI도구를 사용해 편리하게 구성할 수 있으나 재구축이나 수정을 위한 변경사항 추적이 어렵다. 일관적인 배포 환경을 생성하도록 반복 가능하게 하는 다양한 도구들이 있으며 그 중 테라폼을 사용해  AWS 인프라를 구축하고, 나아가 IaC 방식의 인프라 관리법에 숙달하고자 한다.
